# 利用关联规则算法对足球赔率进行数据挖掘的研究

参考论文[《利用关联规则算法对足球赔率进行数据挖掘的研究》](https://www.docin.com/p-602830814.html)

实现了其中的关联规则算法部分，只是挖掘胜率、负率、平率和最终比赛结果之间的关系，找出支持度最高的，并没有全部实现论文中的内容。

简单来说，写着玩的。（事实上想靠它翻身而不得）
